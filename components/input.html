    # input组件

    支持属性如下：

    type: {
      type: String,
      default: "text"
    },
    placeholder: {
      type: String,
      default: ""
    },
    name: {
      type: String,
      default: ""
    },
    disabled: { //是否禁用
      type: Boolean,
      default: false
    },
    value: { //v-model/value
      type: [String, Number]
    },
    maxlength: {
      type: Number
    },
    minlength: {
      type: Number
    },
    clearable: {
      type: Boolean,
      default: false
    },
    showPassword: {
      type: Boolean,
      default: false
    }

    支持如下事件：

      input, clear, blur, focus
